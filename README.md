# SISTEMA-MANEGO

Sistema web para la gestión de servicios de fotografía y video en eventos sociales.  


# Instrucciones para importar la base de datos

Para configurar la base de datos del proyecto, siga los siguientes pasos:

- Inicie XAMPP y active los servicios Apache y MySQL

- En el panel de control de XAMPP, seleccione el botón “Admin” del módulo MySQL.
Esto lo redirigirá automáticamente a phpMyAdmin.

- Una vez dentro de phpMyAdmin, haga clic en “Nueva” para crear una base de datos.
Asigne el nombre: ***manego***
y confirme la creación.

-Con la base de datos creada, seleccione la pestaña “Importar”.

Haga clic en “Seleccionar archivo” y cargue el archivo: ***manego_schema.sql***

-Finalmente, presione “Continuar” para ejecutar la importación.

Una vez completado el proceso, la base de datos quedará lista para funcionar con el proyecto MANEGO.
